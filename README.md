Инструмент для автоматического поиска и классификации обращений с техническими ошибками. Использует комбинацию регулярных выражений и семантического анализа на основе NLP модели.
Скрипт обрабатывает файлы (CSV/Excel) с обращениями пользователей и автоматически определяет, какие из них связаны с техническими ошибками. Особенно полезен для анализа логов поддержки и автоматической категоризации тикетов.

🔍 Как это работает
Два этапа проверки:

Regex поиск - быстрый поиск по ключевым фразам

Нейросетевой анализ - семантическое сравнение с эталонными фразами для более точного поиска

Особенности:
Оптимизированная версия для больших файлов

Прогресс-бар для отслеживания процесса

Автоматическое определение формата файла

Сохраняет контекст найденных ошибок
